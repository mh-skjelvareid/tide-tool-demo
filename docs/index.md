# Tide checking tool 
A small example Python project which checks and visualizes Norwegian tide levels based on the Norwegian Mapping Authority's tide level API. The project is meant for educational purposes, demonstrating tools used in Python development.

## Tide checking API
The information on tides is fetched from the [Norwegian Mapping Authority's API](https://vannstand.kartverket.no/tideapi_en.html)

## Quick start
Call the module for a demo of tide plotting and info on the next high and low tides:

    python -m tide_tool_demo
